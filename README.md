# ContactManagement

**softwares required:**
1. jdk 14
2. maven

**Compile and Run:**
compile command: Go to the project location(...\SpringBootForm) and give below command
                  mvn clean install
run command : Go to the jar file location (..\SpringBootForm\target) and give below command
                  java -jar SpringBootForm-1.0.jar
